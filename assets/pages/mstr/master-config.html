Acceso desde otros dispositivos

Para acceder al Master desde tu celular u otro dispositivo, solo necesitas ingresar tu usuario y contraseña en la pantalla de login. El token de GitHub se puede configurar después desde Configuración → Token de GitHub.

Sin token configurado puedes:
✓ Ver y gestionar reservas ✓ Crear y editar planes, blogs ✓ Ver el calendario ✗ Publicar cambios al sitio web (requiere token)
🔐 El token se guarda cifrado con AES-256 — nunca en texto plano. Es seguro guardarlo en el dispositivo.
Credenciales de GitHub
El token necesita permiso repo (acceso completo a repositorios privados). Se guarda localmente en tu navegador.

Guía de configuración
1 Crea un repositorio privado en GitHub (ej. jans-master-data)
2 Crea una carpeta data/ con 5 archivos JSON vacíos: quotes.json, services.json, plans.json, calendar.json, inventario.json — todos iniciando con [] excepto calendar que inicia con {} e inventario con {"categories":[],"items":[]}
3 Ve a GitHub → Settings → Developer settings → Personal access tokens y genera un token con permiso repo
4 Ingresa tu usuario, repositorio y token arriba, guarda y prueba la conexión

Limpiar Días y Horas Bloqueadas Sin Reservas

Si un día o un horario quedó marcado como "ocupado" pero ya no tiene ninguna reserva confirmada respaldándolo (por ejemplo, si confirmaste y luego regresaste una reserva a pendiente), usa este botón para liberarlo. También te muestra qué horas siguen bloqueadas en este momento.
